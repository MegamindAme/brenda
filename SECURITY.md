# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in Brenda, please report it privately by
emailing [megamindame@gmail.com](mailto:megamindame@gmail.com).

Please include:

- A description of the vulnerability
- Steps to reproduce it
- The affected version(s)
- Any suggested remediation

We will acknowledge receipt within 48 hours and work to resolve the issue
responsibly. Please do not disclose the issue publicly until we have had a
chance to release a fix.
